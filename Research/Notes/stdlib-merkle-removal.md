`std::merkle` is not available in `nargo 1.0.0-beta.19`, so Emtun uses a local helper in [circuit/src/merkle.nr](/C:/Users/HP/KnoxOS/projects/entum/circuit/src/merkle.nr) to compute Merkle roots. The helper uses Poseidon2 consistently for internal node hashing, and the leaf hash in [circuit/src/main.nr](/C:/Users/HP/KnoxOS/projects/entum/circuit/src/main.nr) uses the same Poseidon2 parameterization confirmed during alignment, so the entire tree stays on one hash function end to end.
