# SAP Primitive Manifest
*May 2026*

`sap-primitive-manifest.json` is the machine-readable summary of the Emtun primitive. It records the proof statement, public inputs, private witness boundary, Poseidon2 Merkle parameters, verifier fixture hash, SDK-facing read surfaces, verification commands, and explicit non-goals. The manifest lets reviewers start from one artifact before drilling into the Noir circuit, TypeScript proof tooling, or Solidity verifier surface.
